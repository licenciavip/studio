# **App Name**: CupoCompartido

## Core Features:

- Browse Subscriptions: Display a searchable list of available subscription slots with details like service logo, price, available slots, ratings, and country.
- Subscription Posting: Allow hosts to create subscription listings, specifying the service, number of slots, and desired net earnings, with the system automatically calculating the final price.
- Peruvian Checkout: Enable checkout flow for Peruvian users via bank deposit/transfer (BCP/Interbank). Include a field for users to input the transaction number for verification.
- Latin American Checkout: Integrate a payment gateway (Mercado Pago/PayU) for Latin American users.
- Order and Group Management: Allow customers to view their order statuses (pending/active/dispute) and hosts to manage subscription slots, payment evidences, and held payments.
- Wallet Management: Enable wallet functionality to view held/released balances. Enable withdrawals via Yape (Peru) or wallet (LatAm).
- Dispute Resolution Tool: Allow users to open disputes with supporting evidence. An LLM is incorporated as a tool to provide a recommendation for proportional refunds in case of early removal from the subscription.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2), reflecting trust and connection in a shared marketplace. This references the financial components of the app as well as the collaborative element, so a professional color is more appropriate than a friendly color.
- Background color: Light blue (#E5F5FF), offering a clean and calming backdrop.
- Accent color: Yellow-orange (#F9A03F), providing highlights for calls to action. 
- Body font: 'PT Sans', a humanist sans-serif that provides a modern yet slightly warm feel, suitable for various content lengths.
- Headline font: 'Playfair', a modern serif offering an elegant feel to headlines; pair with 'PT Sans' for longer text.
- Use simple, clear icons to represent subscription services and transaction statuses.
- Employ a clean and intuitive layout that prioritizes clear display of subscription information and easy navigation through checkout processes and management panels.
